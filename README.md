# Asset Inventory Management System

A compact, full-stack starter for asset inventory management and reporting.

## Quick start

1. Clone the repository:

```bash
git clone https://github.com/sjmanyarkiy/asset-inventory-management-system.git
cd asset-inventory-management-system
```

1. Backend (development):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements.txt
export FLASK_APP=backend.wsgi:app
flask run --port 5000
```

1. Frontend (development):

```bash
cd frontend
npm ci
npm start
```

## Running tests

- Backend: `PYTHONPATH=. pytest -q backend/tests`
- Frontend: run tests from `frontend/` with `npm test`

## Render production deployment

This repository now includes a root `render.yaml` Blueprint for the easiest production setup.

- **Backend (Web Service)**
  - Root directory: `backend`
  - Build command: `pip install -r requirements.txt`
  - Start command: `gunicorn wsgi:app`

- **Frontend (Static Site)**
  - Root directory: `frontend`
  - Build command: `npm run build`
  - Publish directory: `dist`

### Required environment values on Render

- Backend:
  - `DATABASE_URL`
  - `CORS_ORIGINS` (set this to your frontend Render URL)
  - `SECRET_KEY` and `JWT_SECRET_KEY` can be auto-generated by Render Blueprint

- Frontend:
  - `VITE_API_BASE_URL` (set this to your backend Render URL, e.g. `https://<backend>.onrender.com`)

Render handles hosting, SSL, and scaling based on your selected plan.
